# Taiwan Job Market Analysis

使用 Python 爬取 1111 人力銀行大數據職缺資料，清洗薪資欄位，並使用 Tableau 製作台灣工作機會分布與薪資分析 Dashboard。

## Project Features

- 使用 Requests + BeautifulSoup 爬取職缺資料
- 使用 OpenPyXL 輸出 Excel
- 清洗月薪、年薪、時薪、面議等薪資格式
- 建立薪資上限、薪資下限、薪資平均欄位
- 使用 Tableau 製作台灣地圖分布圖與薪資排名圖

## Tools

- Python
- BeautifulSoup
- OpenPyXL
- Excel
- Tableau Public
